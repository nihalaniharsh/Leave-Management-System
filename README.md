# Leave Management System

The Leave Management System is a full-stack web application that allows employees to apply for leave and administrators to manage leave requests efficiently. The system provides a structured way to track leave applications, approvals, and records.

This project demonstrates full-stack development using modern technologies.

---

## Features

### Employee Features
- Apply for leave
- View leave history
- Check leave status (approved / pending / rejected)

### Admin Features
- View all leave requests
- Approve or reject leave applications
- Manage employee leave records

---

## Installation

### Clone the repository


git clone https://github.com/nihalaniharsh/Leave-Management-System.git


### Navigate to the project folder


cd Leave-Management-System


---

## Backend Setup


cd backend
npm install
npm start


Backend will run on:


http://localhost:5000


---

## Frontend Setup

Open another terminal:


cd frontend
npm install
npm run dev


Frontend will run on:


http://localhost:5173
